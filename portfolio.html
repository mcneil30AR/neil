Me integré a la Agencia de Diseño Altera como Web Developer, donde dimos vida a proyectos generados en la agencia que era hora de mostrar como catálogo de servicios. El reto fue seleccionar y crear un espacio que luciera atractivo y juvenil, pero que también dejara en claro la experiencia de esta gran agencia tampiqueña.
El sitio web de la empresa Kmeisa nos permitió trabajar en la plataforma Webflow y aprender sobre la marcha para ofrecer un cambio en la forma de generar páginas web más dinámicas.
El reto de este proyecto fue generar una plantilla desde cero para WordPress. El cliente requería que fuera en esta plataforma y no deseaba una plantilla personalizada del mercado. Basándonos en el diseño de la agencia y utilizando programación en PHP, HTML, JavaScript y CSS, creamos esta propuesta a medida para la empresa TLA de Altamira.
El proyecto del sitio web de Grupo Freedom es, sin duda, uno de mis favoritos, ya que explotamos la imaginación y las características de la plataforma Webflow y JavaScript para hacer realidad este diseño complejo pero divertido.
